# 🚀 30 Days of Terraform

Welcome to my **#30DaysOfTerraform** learning journey.

I’ve started learning Terraform by following the amazing content from **Piyush Sachdeva’s YouTube channel**, and this repository will document my daily hands-on practice, concepts, and real-world infrastructure experiments.

---

## 📚 Why This Series?

Infrastructure as Code (IaC) is foundational for modern DevOps and Cloud Engineering.

Instead of clicking through cloud consoles, Terraform allows us to:

- Define infrastructure in code
- Version control infrastructure
- Automate deployments
- Maintain consistency across environments
- Work across multiple cloud providers

This 30-day journey is about building strong fundamentals before moving into advanced production-level use cases.

---

## 🎯 Goals of This Challenge

- Understand Terraform core concepts
- Master Terraform workflow (`init`, `plan`, `apply`, `destroy`)
- Learn providers and resource management
- Build real AWS infrastructure using Terraform
- Implement best practices
- Understand state management
- Explore modules and reusability
- Learn how Terraform scales in SaaS & FinTech environments
